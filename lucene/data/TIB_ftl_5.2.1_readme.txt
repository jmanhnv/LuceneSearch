Introduction 

TIBCO FTL(R) is an application-to-application messaging system
designed for low latency and high performance.

TIBCO is proud to announce the latest release of TIBCO FTL(R).
This release is the latest in a long history of TIBCO products
that leverage the power of the Information Bus(R) to enable
event-driven IT environments.  To find out more about how TIBCO
FTL and other TIBCO products are powered by TIB(R) technology,
please visit us at www.tibco.com.

Minimum Software Requirements

   Third-party Software:
     Sun Java Development Kit 1.8.x
     Microsoft Visual Studio 2015

Additional Information 

   64-bit hardware, operating system, compiler, linker and JDK
   are required:
   
   Java applications require JDK 1.8 (64-bit).
   
   Linux applications require the 2.6 kernel and glibc 2.5 (or
   later).
   
   Windows applications require Visual Studio 2015 (VC14).
   
   In order to use the RDMA transport type, you must obtain the
   latest RDMA and OFED libraries from your hardware vendor.
